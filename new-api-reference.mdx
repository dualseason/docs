Xcompute 提供 OpenAI 兼容接口。你可以使用同一个 API Key 调用文本、图片、音频和视频模型。

## 快速开始

### 1. 获取 API Key

登录 [Xcompute 控制台](https://xcompute.us/keys)，进入 **个人中心 → API Keys** 创建密钥。

<Frame>
  ![创建 API Key](/images/image-23.png)
</Frame>

<Frame>
  ![选择 API Key 分组](/images/image-24.png)
</Frame>

创建密钥时需要选择分组：

| 使用场景 | 推荐分组 | 说明 |
| --- | --- | --- |
| GPT 文本模型 | GPT 福利渠道 0.9 折 | 按官方价格的 9% 计费 |
| Claude 文本模型 | Kiro 1.8 折 | 适合大部分 Claude 使用场景 |
| 图片、视频模型 | `default` | 图片和视频模型使用默认分组 |

<Info>
  如果页面受翻译插件影响而出现错位或乱码，请先关闭翻译插件后刷新。
</Info>

### 2. 配置接口

OpenAI 兼容客户端只需要配置 Base URL、API Key 和模型 ID。

| 配置项 | 内容 |
| --- | --- |
| 海外 Base URL | `https://xcompute.us/v1` |
| 国内加速 Base URL | `https://intl.dualseason.com/v1` |
| API Key | 控制台创建的密钥 |
| Model ID | 模型广场复制的标准模型 ID |

<Frame>
  ![填写 API 配置](/images/image-25.png)
</Frame>

### 3. 快速测试

```bash
curl https://intl.dualseason.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-5.6-sol",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

<Warning>
  请把 `YOUR_API_KEY` 替换为你在控制台创建的真实 API Key。不要把密钥提交到公开代码仓库。
</Warning>

## API 调用示例

### 聊天补全

```bash
curl https://xcompute.us/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-5.5",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "Hello!"}
    ],
    "stream": false
  }'
```

### 图像生成

```bash
curl https://xcompute.us/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "A cute cat playing piano",
    "n": 1,
    "size": "1024x1024",
    "quality": "high"
  }'
```

## 常见问题

### 如何选择 Base URL？

- 海外用户使用 `https://xcompute.us/v1`。
- 国内用户使用 `https://intl.dualseason.com/v1` 加速入口。

### 如何选择分组？

- GPT 文本模型优先选择 GPT 福利渠道 0.9 折。
- Claude 日常使用选择 Kiro 1.8 折。
- Claude 蒸馏或更高智能需求选择官 Key 4.9 折。
- 图片和视频模型选择 `default`。

### 图像生成失败怎么办？

`gpt-image-2` 可能受 OpenAI 风控影响而短期资源不足。请稍后重试；持续失败时在官方群内反馈，管理员会检查渠道资源。

### 如何计算费用？

```text
总费用 = 输入单价 × 输入 Token 数 / 1,000,000
       + 输出单价 × 输出 Token 数 / 1,000,000
```

模型表中的价格已按对应渠道折扣换算，无需再次乘渠道倍率。
