这份文档只做新版接口入口。详细内容按类别拆到独立页面里。

## 基础信息

- `Base URL`：`https://xcompute.us`
- 控制台接口前缀：`/api`
- 模型调用接口前缀：`/v1`、`/v1beta`
- 控制台鉴权：登录后的 session，或用户 access token
- 模型调用鉴权：`Authorization: Bearer YOUR_API_KEY`
