Claude Code 教程 — 将 Anthropic 的终端编程助手对接Xcompute。支持代码理解、多文件编辑，集成 VS Code 与 JetBrains IDE。

**ℹ️  项目介绍**

直接在你的终端中释放 Claude 的强大能力。瞬间搜索百万行代码库。将耗时数小时的工作流程化为一条命令。你的工具，你的工作流，你的代码库，以思维速度进化。
