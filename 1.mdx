这篇指南介绍如何把 Claude Code 接入 Xcompute，方便你在终端编程工作流中统一使用模型与密钥。

## 项目介绍

直接在你的终端中释放 Claude 的强大能力。瞬间搜索百万行代码库。将耗时数小时的工作流程化为一条命令。你的工具，你的工作流，你的代码库，以思维速度进化。

- 官方主页：[https://www.anthropic.com/claude-code](https://www.anthropic.com/claude-code)

## 接入思路

<Steps>
  <Step title="准备独立 API Key">
    建议专门为 Claude Code 创建一个 API Key，便于和应用侧调用分开管理。
  </Step>
  <Step title="配置自定义接口地址">
    在 Claude Code 的模型提供方配置中，使用 Xcompute 的接口地址和对应密钥。
  </Step>
  <Step title="用真实仓库做一次验证">
    让 Claude Code 在本地仓库里执行一次搜索、解释或编辑任务，确认响应速度与模型行为符合预期。
  </Step>
</Steps>
