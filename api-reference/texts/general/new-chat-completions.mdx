---
title: '通用对话接口'
description: '使用 OpenAI Chat Completions 兼容格式调用文本生成模型。'
api: 'POST /v1/chat/completions'
---

- 统一的对话 API 接口，支持所有文本生成模型
- 通过 `model` 参数选择不同的 AI 模型
- 兼容 OpenAI Chat Completions API 格式

## 支持模型

| 系列 | 支持的模型 ID |
| --- | --- |
| GPT | `gpt-5.6-sol`、`gpt-5.6-terra`、`gpt-5.6-luna`、`gpt-5.5`、`gpt-5.4` |
| Claude | `claude-fable-5`、`claude-sonnet-5`、`claude-opus-4-8` |
| Gemini | `gemini-3.1-pro-high`、`gemini-3.1-pro-preview`、`gemini-3.1-pro-preview-think`、`gemini-3.5-flash` |
| Grok | `grok-4.5`、`grok-4.3` |
| GLM | `glm-5`、`glm-5-turbo`、`glm-5.1`、`glm-5.2`、`glm-5v-turbo` |
| Kimi | `kimi-k2.5`、`kimi-k2.6`、`kimi-k2.7-code` |
| DeepSeek | `deepseek-v3.2`、`deepseek-v4-flash`、`deepseek-v4-pro` |
| Qwen | `qwen3.5-plus`、`qwen3.6-plus`、`qwen3.7-plus`、`qwen3.7-max` |
| MiniMax | `minimax-m3`、`minimax-m2.5`、`minimax-m2.7`、`MiniMax-M2.7-highspeed` |
| Mimo | `mimo-v2.5`、`mimo-v2.5-pro`、`mimo-v2-pro`、`mimo-v2-omni` |

<Info>
  模型可用性可能随渠道调整，请以 [模型广场](https://xcompute.us/models) 中显示的模型 ID 为准。
</Info>

<RequestExample>
  ```bash cURL theme={null}
  curl --request POST \
    --url https://xcompute.us/v1/chat/completions \
    --header 'Authorization: Bearer <token>' \
    --header 'Content-Type: application/json' \
    --data '{
      "model": "gpt-5.6-sol",
      "messages": [
        {
          "role": "user",
          "content": "你好，请介绍一下你自己"
        }
      ]
    }'
  ```
</RequestExample>

<ParamField body="model" type="string" required>
  模型 ID，例如 `gpt-5.6-sol`。
</ParamField>

<ParamField body="messages" type="array" required>
  对话消息数组。
</ParamField>

<ResponseExample>
  ```json 200 theme={null}
  {
    "id": "chatcmpl_xxx",
    "object": "chat.completion"
  }
  ```
</ResponseExample>
