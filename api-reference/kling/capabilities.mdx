Xcompute 提供可灵原生兼容接口和统一多媒体接口。可灵任务均为异步任务：提交成功后获得任务 ID，再通过对应的查询接口获取结果。

<Info>
  接口能力会受到模型版本和上游渠道影响。模型与功能的实时可用性请以 [模型广场](https://xcompute.us/models) 为准。
</Info>

## 接入信息

| 配置项 | 内容 |
| --- | --- |
| Base URL | `https://xcompute.us` |
| 鉴权方式 | `Authorization: Bearer YOUR_API_KEY` |
| 音频前缀 | `/kling/v1/audio` |
| 图片前缀 | `/api/klingai/v1` |
| OpenAI Speech 兼容入口 | `POST /v1/audio/speech` |

## 支持模型

可灵渠道支持以下模型 ID：

- `kling-v1`
- `kling-v1-5`
- `kling-v1-6`
- `kling-v2-5-turbo`
- `kling-v2-6`
- `kling-v3`
- `kling-v3-omni`

## 能力矩阵

| 类别 | 能力 | 状态 | 主要输入 |
| --- | --- | --- | --- |
| 音频 | 文本生成音频 | 支持 | `prompt` |
| 音频 | 视频生成音频 | 支持 | `video_id` 或 `video_url` |
| 语音 | 文本转语音 TTS | 支持 | `text`、`voice_id`、语言、语速 |
| 图片 | 图片生成 | 支持 | `prompt`、分辨率、数量 |
| 图片 | 多图生图 | 支持 | 主体图、场景图、风格图 |
| 图片 | 智能扩图 | 支持 | 图片、四个方向的扩展比例 |
| 语音 | 音色克隆 | 暂不支持 | 当前没有独立的音色克隆任务接口 |

## 音频与语音接口

### 文本生成音频

```bash
curl https://xcompute.us/kling/v1/audio/text-to-audio \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model_name": "kling-v3",
    "prompt": "雨夜街道的环境声，远处偶尔传来汽车经过的声音"
  }'
```

### 视频生成音频

```bash
curl https://xcompute.us/kling/v1/audio/video-to-audio \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model_name": "kling-v3",
    "video_url": "https://example.com/input.mp4"
  }'
```

### 文本转语音 TTS

OpenAI 请求字段兼容入口：

```bash
curl https://xcompute.us/v1/audio/speech \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-v3",
    "input": "欢迎使用 Xcompute 可灵语音合成接口。",
    "voice": "YOUR_VOICE_ID",
    "speed": 1.0,
    "language": "zh"
  }'
```

该入口对可灵返回异步任务 JSON，不直接返回音频二进制流。

可灵原生入口：

```bash
curl https://xcompute.us/kling/v1/audio/tts \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model_name": "kling-v3",
    "text": "欢迎使用 Xcompute 可灵语音合成接口。",
    "voice_id": "YOUR_VOICE_ID",
    "voice_language": "zh",
    "voice_speed": 1.0
  }'
```

| TTS 参数 | 类型 | 说明 |
| --- | --- | --- |
| `text` | string | 必填，需要合成的文本 |
| `voice_id` | string | 上游支持的音色 ID |
| `voice_language` | string | 语音语言 |
| `voice_speed` | number | 语速设置，范围以当前上游模型为准 |

<Warning>
  当前接口支持选择已有音色，但不提供独立的音色克隆任务。不要将 `voice_id` 选择与音色克隆混为一谈。
</Warning>

## 图片接口

| 能力 | 提交接口 | 查询接口 |
| --- | --- | --- |
| 图片生成 | `POST /api/klingai/v1/images/generations` | `GET /api/klingai/v1/images/generations/{task_id}` |
| 多图生图 | `POST /api/klingai/v1/images/multi-image2image` | `GET /api/klingai/v1/images/multi-image2image/{task_id}` |
| 智能扩图 | `POST /api/klingai/v1/images/editing/expand` | `GET /api/klingai/v1/images/editing/expand/{task_id}` |

图片任务支持的主要字段包括：

- `image_list`、`image_reference`
- `subject_image_list`
- `scene_image`、`style_image`
- `image_fidelity`、`human_fidelity`
- `resolution`、`n`
- `up_expansion_ratio`、`down_expansion_ratio`
- `left_expansion_ratio`、`right_expansion_ratio`

## 查询任务

提交成功后会返回公开任务 ID：

```json
{
  "id": "task_01KXXXXXXXXXXXXXXXXXXXX",
  "status": "queued"
}
```

也可以通过统一任务查询接口获取状态：

```bash
curl https://xcompute.us/v1/tasks/task_01KXXXXXXXXXXXXXXXXXXXX \
  -H "Authorization: Bearer YOUR_API_KEY"
```

任务常见状态包括 `queued`、`in_progress`、`completed` 和 `failed`。任务完成后，响应中的 `metadata.url` 或结果字段会包含音频或图片地址。

## 使用建议

1. 将用户侧任务 ID 与返回的 `task_id` 一并保存，方便重试和排查。
2. 对音频和图片结果进行持久化，不要长期依赖上游临时 URL。
3. 高级能力的字段组合依赖具体模型。切换模型时重新检查输入限制。
4. 生产环境优先使用任务查询或回调，不要高频无间隔轮询。
