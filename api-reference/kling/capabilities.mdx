Xcompute 当前接入 **17 项可灵能力**。每项能力都有独立接口页面。

<Info>
  所有生成类接口均为异步任务。提交成功后，使用对应的原生查询接口或[统一任务查询接口](/api-reference/tasks/new-status)获取结果。
</Info>

## 接入信息

| 配置项 | 内容 |
| --- | --- |
| Base URL | `https://xcompute.us` |
| 鉴权方式 | `Authorization: Bearer YOUR_API_KEY` |
| 视频、数字人、音频前缀 | `/kling/v1` |
| 图片前缀 | `/api/klingai/v1` |

## 支持模型

`kling-v1`、`kling-v1-5`、`kling-v1-6`、`kling-v2-5-turbo`、`kling-v2-6`、`kling-v3`、`kling-v3-omni`

模型与能力的实时可用性取决于当前渠道，请以[模型广场](https://xcompute.us/models)为准。
