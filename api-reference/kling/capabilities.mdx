Xcompute 提供可灵原生兼容接口和统一多媒体接口。可灵任务均为异步任务：提交成功后获得任务 ID，再通过对应的查询接口获取结果。

<Info>
  接口能力会受到模型版本和上游渠道影响。模型与功能的实时可用性请以 [模型广场](https://xcompute.us/models) 为准。
</Info>

## 接入信息

| 配置项 | 内容 |
| --- | --- |
| Base URL | `https://xcompute.us` |
| 鉴权方式 | `Authorization: Bearer YOUR_API_KEY` |
| 视频与音频前缀 | `/kling/v1` |
| 图片前缀 | `/api/klingai/v1` |
| 统一视频接口 | `POST /v1/videos/generations` |
| OpenAI Speech 兼容入口 | `POST /v1/audio/speech` |

## 支持模型

可灵渠道适配器支持以下模型 ID：

- `kling-v1`
- `kling-v1-5`
- `kling-v1-6`
- `kling-v2-5-turbo`
- `kling-v2-6`
- `kling-v3`
- `kling-v3-omni`

部分统一视频渠道还提供 `kling-video-o1`。调用前请确认所选渠道已经配置对应模型。

### 常用视频参数

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `model_name` | string | 可灵模型 ID；也兼容 `model` |
| `prompt` | string | 视频内容描述 |
| `negative_prompt` | string | 不希望出现的内容 |
| `image` | string | 首帧图片 URL 或 Base64 |
| `image_tail` | string | 尾帧图片 URL 或 Base64 |
| `mode` | string | 生成模式，例如 `std` 或 `pro` |
| `duration` | string | 视频时长，具体范围取决于模型 |
| `aspect_ratio` | string | 画面比例，例如 `16:9`、`9:16`、`1:1` |
| `sound` | string | Omni 或支持原生音频的模型声音设置 |
| `camera_control` | object | 平移、倾斜、旋转和变焦等镜头控制 |
| `callback_url` | string | 任务状态回调地址 |
| `external_task_id` | string | 调用方自定义任务标识 |

## 音频与语音接口

### 文本生成音频

```bash
curl https://xcompute.us/kling/v1/audio/text-to-audio \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model_name": "kling-v3",
    "prompt": "雨夜街道的环境声，远处偶尔传来汽车经过的声音"
  }'
```

### 视频生成音频

```bash
curl https://xcompute.us/kling/v1/audio/video-to-audio \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model_name": "kling-v3",
    "video_url": "https://example.com/input.mp4"
  }'
```

### 文本转语音 TTS

OpenAI 请求字段兼容入口：

```bash
curl https://xcompute.us/v1/audio/speech \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-v3",
    "input": "欢迎使用 Xcompute 可灵语音合成接口。",
    "voice": "YOUR_VOICE_ID",
    "speed": 1.0,
    "language": "zh"
  }'
```

该入口对可灵返回异步任务 JSON，不直接返回音频二进制流。

可灵原生入口：

```bash
curl https://xcompute.us/kling/v1/audio/tts \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model_name": "kling-v3",
    "text": "欢迎使用 Xcompute 可灵语音合成接口。",
    "voice_id": "YOUR_VOICE_ID",
    "voice_language": "zh",
    "voice_speed": 1
  }'
```

| TTS 参数 | 类型 | 说明 |
| --- | --- | --- |
| `text` | string | 必填，需要合成的文本 |
| `voice_id` | string | 上游支持的音色 ID |
| `voice_language` | string | 语音语言 |
| `voice_speed` | integer | 语速设置，范围以当前上游模型为准 |

<Warning>
  当前接口支持选择已有音色，但不提供独立的音色克隆任务。不要将 `voice_id` 选择与音色克隆混为一谈。
</Warning>

## 口型同步与数字人

高级口型同步可使用以下字段组合：

- `video_id` 或视频素材
- `audio_id` 或 `sound_file`
- `face_choose`，指定需要同步的人脸
- `session_id`，关联人脸识别任务

图片数字人接口还支持 `image_url` 和 `character_orientation`。不同渠道对素材格式、文件大小和时长可能有额外限制。

## 图片接口

| 能力 | 提交接口 | 查询接口 |
| --- | --- | --- |
| 图片生成 | `POST /api/klingai/v1/images/generations` | `GET /api/klingai/v1/images/generations/{task_id}` |
| 多图生图 | `POST /api/klingai/v1/images/multi-image2image` | `GET /api/klingai/v1/images/multi-image2image/{task_id}` |
| 智能扩图 | `POST /api/klingai/v1/images/editing/expand` | `GET /api/klingai/v1/images/editing/expand/{task_id}` |

图片任务支持的主要字段包括：

- `image_list`、`image_reference`
- `subject_image_list`
- `scene_image`、`style_image`
- `image_fidelity`、`human_fidelity`
- `resolution`、`n`
- `up_expansion_ratio`、`down_expansion_ratio`
- `left_expansion_ratio`、`right_expansion_ratio`

## 查询任务

提交成功后会返回公开任务 ID：

```json
{
  "id": "task_01KXXXXXXXXXXXXXXXXXXXX",
  "status": "queued"
}
```

使用与提交能力对应的 GET 接口查询。也可以通过统一任务查询接口获取状态：

```bash
curl https://xcompute.us/v1/tasks/task_01KXXXXXXXXXXXXXXXXXXXX \
  -H "Authorization: Bearer YOUR_API_KEY"
```

任务常见状态包括 `queued`、`in_progress`、`completed` 和 `failed`。任务完成后，响应中的 `metadata.url` 或结果字段会包含视频、音频或图片地址。

## 使用建议

1. 先用较短时长、标准模式验证提示词和素材，再提高分辨率或切换专业模式。
2. 将用户侧任务 ID 与返回的 `task_id` 一并保存，方便重试和排查。
3. 对视频、音频和图片结果进行持久化，不要长期依赖上游临时 URL。
4. 高级能力的字段组合依赖具体模型。切换模型时重新检查素材限制。
5. 生产环境优先使用任务查询或回调，不要高频无间隔轮询。
