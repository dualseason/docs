---
title: '用户登录'
api: 'POST /api/user/login'
---

- 账号密码登录
- 登录成功后写入 session
- 如果账号启用 2FA，会继续返回二次验证流程

<RequestExample>
  ```bash cURL theme={null}
  curl --request POST \
    --url https://xcompute.us/api/user/login \
    --header 'Content-Type: application/json' \
    --data '{
      "username": "YOUR_USERNAME",
      "password": "YOUR_PASSWORD"
    }'
  ```
</RequestExample>
