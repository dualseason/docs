</Info>

## 接入说明

<Steps>
  <Step title="获取 API Key">
    前往 [API 密钥页面](https://claw.dualseason.com/console/token) 创建并复制你的密钥。
  </Step>
  <Step title="选择接口与模型">
    根据你的业务类型选择文本、图像、视频或音频接口，并在请求中指定对应模型。
  </Step>
  <Step title="发送请求并轮询结果">
    文本接口通常同步返回；图像和视频接口通常返回 `task_id`，需要再调用任务状态接口查询结果。
  </Step>
</Steps>

<Tip>
  如果你已经兼容 OpenAI 风格请求格式，优先从 `/v1/chat/completions`、`/v1/images/generations` 和 `/v1/videos/generations` 开始接入。
</Tip>
