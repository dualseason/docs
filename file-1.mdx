## **🎯 第一步：理解木瓜AI - 一站式AI大模型API聚合中转平台**

**💡 什么是API中转？**\
我们的核心服务是“API中转”。简单来说，您只需要将其他工具（如 ChatGPT 客户端）中的 API 请求地址，从官方地址换成我们提供的地址即可。

**原地址**：`https://api.openai.com` ➔ **Xcompute 地址**：`https://xcompute.us`

**核心优势：**

✅ 多渠道和计费优化，费用远低于官方价格

✅ 一个接口访问所有主流AI模型

✅ 多线路选择，网络连接更稳定

> 掌握Xcompute独有的“渠道分组”和“计费模式”，将极大优化您的成本。

### **🧠 核心一：巧选“渠道分组”**

您可以将“渠道分组”理解为不同舱位的机票，在创建令牌时即可选择：

`官方渠道 `**分组（商务舱）**：速度快、稳定性高，适合需要高质量响应的严肃任务。

`特惠渠道 `**分组（经济舱）**：价格最低，适合对成本极度敏感的个人项目。

`default `**分组（智能优选）⭐**：新手推荐！系统会自动为您选择合适的渠道(官方渠道优先)。

### **💸 核心二：看场景选“计费模式”**

我们支持两种计费模式，选对能省一大笔钱！

🚕 **按量计费（默认）**：像打车，按字数计费。适合日常短对话、快速问答等场景。

🚇 **按次计费**：像地铁票，一口价。适合写长文章、分析报告等长内容场景。

> 💡 **黄金法则**：单次请求\+回复总字数超过20000字时，使用“按次计费”更划算！

## **🔑 第二步：开始简单使用**

**💡 什么是令牌？**\
“令牌”（Token / API Key）是您调用AI的唯一凭证，就像一把专属的数字钥匙。有了它，您就可以在任何支持自定义接口的AI工具中使用我们的服务。

### **📝 创建您的令牌 (🟢 必做步骤)**

1. **前往令牌管理页面**：点击“添加令牌”按钮。\
   _(推荐命名示例：我的日常使用Key / 工作专用Token)_
2. **获取密钥 (⚠️ 重要)**：创建后系统会生成以 `sk-` 开头的密钥。

> **重要提醒**：密钥生成后请妥善保存，一旦泄漏，可能造成财产损失！

### **🚀 选择使用方式 (🔵 多种选择)**

**💬 站内聊天页面（新手推荐）**\
这是最简单的方式，登录后直接在网页上与 AI 对话。

- √ 无需下载安装

- √ 支持所有AI模型

**⚙️ 第三方工具（高级）**\
几乎所有支持自定义接口的AI客户端都适用。

**API Key**: `sk-...` (您刚刚生成的密钥)

**API Base URL**: `https://xcompute.us`\
_(支持查看推荐客户端工具列表及编程工具列表)_

## **🤖 第三步：选择适合的AI模型**

打开模型广场，里面有的模型都能选择

### 文字生成模型

- **GPT-5.5**: 目前最强大的对话和文本生成能力，并且兼具性价比的一款模型，适合写代码做任务
- **Claude-opus-4-7**: Anthropic 的高性能对话模型，比上一代4-6要更加便宜，方案架构能力突出
- **Gemini 3.1**: Google 的多模态大语言模型，优点和价格都不突出，不推荐使用
- **GLM 5.1**：国产智普清言大模型，优点稳定，性能和价格不突出，不推荐，适合备用
- **Kimi k2.6**：国产月之暗面大模型，优点速度快，代码哗哗写，写小工具比较适合，适合快速产出
- **MiniMax M2.7**：广告达人，不推荐，前端页面还不错，在opencode中是常驻免费大模型

### 图像生成模型

- **GPT-image-2**: 高质量图像生成，米粒刻字
- **Nano Banana 2**: 专业生图，比**_GPT-image-2_** 出的时间早，所以只有画质比**_GPT-image-2_** 好

### 视频生成模型

- **Seedance 2.0系列**: 工业级别视频生成模型，缺点就是贵，而且字节跳动天天涨价
- **Veo 3.1**: 谷歌的Gemini模型
- **Grok-4.2**: 马斯克的视频模型
