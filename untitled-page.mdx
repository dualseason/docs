## 1. 了解 Codex 是什么？有什么用处？

Codex 是 OpenAI 推出的 AI 编程智能体（Coding Agent）—— 能听懂人话，按你的指令行事，会自己修改代码的 AI。

**用处**分为两大类：

**程序员用：**
- 用看得懂的中文回答他做了什么
- 按你的需求操控电脑进行数据修改与代码编写
- 支持多任务操作，可开多个窗口一起工作

**非程序员用：**
- 帮你整理桌面文件
- 辅助完成每日工作，减轻工作压力
- 需要成品时，叫 Codex 帮你生成一个，简单快捷

## 2. 如何安装 Codex

### ① 程序员用 — CLI（命令行格式）

#### Mac 系统

**方式一：npm 安装**（如果没有 npm，先安装 Node.js，自带 npm）

<Frame>
  <img src="/images/image-19.png" alt="Node.js 下载页面" />
</Frame>

Node.js 下载地址（需 VPN）：[https://nodejs.org/en](https://nodejs.org/en)

```bash
npm install -g @openai/codex
```

**方式二：Homebrew 安装**

```bash
brew -v
brew install --cask codex
```

#### Windows 系统

**WSL 安装方式**（以管理员身份运行 PowerShell）：

```powershell
wsl --install -d Ubuntu-24.04
```

重启进入 WSL 后，安装 Node.js 和 Codex：

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc   # 或 ~/.zshrc
nvm install 22
npm install -g @openai/codex
```

#### Linux 系统

**方式一：npm 安装**

```bash
npm install -g @openai/codex
```

**方式二：二进制离线安装**

```bash
# x86_64
curl -L https://github.com/openai/codex/releases/download/rust-v0.131.0/codex-x86_64-unknown-linux-musl.tar.gz | tar -xz
sudo mv codex /usr/local/bin/

# ARM64（树莓派、AWS Graviton 等）
# 将文件名中的 x86_64 替换为 aarch64 即可
```

安装完成后验证：

```bash
codex --version
```

如果弹出版本号（如 `0.xxx.0`）即安装成功。

首次在终端输入 `codex` 运行，会有两个选项：
1. **ChatGPT 账号** — 推荐新手，有免费额度
2. **OpenAI API Key** — 适合开发者，可通过 CC Switch 中转

## 3. 配置 Codex

这里我们使用 **CC Switch** 来配置 Codex。

### 什么是 CC Switch？

<Info>
  CC Switch 是一款面向 AI 编程 CLI 用户的跨平台桌面配置管理工具。它能管理 Claude Code、Codex、Gemini CLI、OpenCode、OpenClaw 等 AI 工具，并查看各工具的 Token 用量。简单说：CC Switch 是一位管家，负责看管与配置你手里的 AI 工具。
</Info>

**下载 CC Switch**：[https://github.com/farion1231/cc-switch/releases](https://github.com/farion1231/cc-switch/releases)

下载完成后打开，进入主界面：

<Frame>
  <img src="/images/image-21.png" alt="CC Switch 主界面" />
</Frame>

要配置 Codex，点击列表中从左往右的第 3 个（上面两个为官网自带，不能使用；下面两个为自定义配置）：

<Frame>
  <img src="/images/9ed1f43087008d870e8f6f91cc4c3353.png" alt="CC Switch 供应商列表" />
</Frame>

### 配置步骤

<Steps>
  <Step title="添加新供应商">
    点击右上角的橙色 **"+"** 号，向下滑动页面开始配置：

    <Frame>
      <img src="/images/8df0d5be7717ec139fa0c894e56982c2.png" alt="CC Switch 配置表单" />
    </Frame>
  </Step>

  <Step title="填写信息">
    | 字段 | 说明 |
    | --- | --- |
    | **供应商名称** | 可随意填写 |
    | **API Key** | 以 `sk-` 开头的密钥 |
    | **请求地址** | 中转站地址，格式为 `https://xxx.com/v1` |
  </Step>

  <Step title="保存">
    填写完成后点击 **添加** 即可完成配置。
  </Step>
</Steps>
