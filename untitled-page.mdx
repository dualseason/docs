**ℹ️  项目介绍**

OpenClaw 是一个开源、自托管的个人 AI 助手平台，将消息应用连接到运行在你自己硬件上的 AI 代理。专为开发者和高级用户设计，无需交出数据控制权即可拥有自主 AI 助手。

- 官方主页：[**<u>https://openclaw.ai</u>**](https://openclaw.ai)
- 项目文档：[**<u>https://docs.openclaw.ai</u>**](https://docs.openclaw.ai)
- GitHub：[**<u>https://github.com/openclaw/openclaw</u>**](https://github.com/openclaw/openclaw)

OpenClaw 完全开源，你可以在 [**<u>OpenClaw 的 GitHub 仓库</u>**](https://github.com/openclaw/openclaw) 浏览源码、提交 Issue 或参与贡献。本教程涵盖安装、配置，以及将 OpenClaw 对接 Xcompute 的 API 的完整步骤。
