Xcompute的 AI API 平台概览，包含国际站、加速入口和国内站说明。 平台提供文本、图像、音频和视频能力。你可以用一套兼容 OpenAI 的接口，把模型能力快速集成到你的产品中。

| 站点 | 域名 | 说明 |
| --- | --- | --- |
| 国际站 | `xcompute.us` | 香港服务器，面向国际网络环境。（需要科学上网） |
| 加速入口 | `intl.dualseason.com` | 国内访问国际站的加速转发入口。（国内环境访问） |
| 国内站 | `xcompute.dualseason.com` | 国内独立站点，与国际站完全隔离。（不推荐访问） |

## 为什么选择 Xcompute中转站

1. 一个网站包含所有主流大模型的接入，不需要对接繁琐的各大官网和充值。
2. 有专业客服和售后技术团队，相比大模型厂商官网他们没有专人服务。
3. 价格比官网价格更低，国内模型有集采优惠，国外模型有技术手段降价。
4. 有一整套交付软件配套：Codex、Claude code、Opencode、龙虾、爱马仕。
5. 有自己的软件智能体：移动端APP、无限画布工作流。
