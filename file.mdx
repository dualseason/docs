Dualseason 的 AI API 平台提供文本、图像、音频和视频能力。你可以用一套兼容 OpenAI 的接口，把模型能力快速集成到你的产品中。

<Info>
  `xcompute.us` 是香港国际站。`intl.dualseason.com` 是国内访问国际站的加速入口。`xcompute.dualseason.com` 是国内独立站。三者账号、数据和计费完全隔离。
</Info>

## 站点说明

| 站点 | 域名 | 说明 |
| --- | --- | --- |
| 国际站 | `xcompute.us` | 香港服务器，面向国际网络环境。 |
| 国际加速入口 | `intl.dualseason.com` | 国内访问国际站的加速转发入口。 |
| 国内站 | `xcompute.dualseason.com` | 国内独立站点，与国际站完全隔离。 |

## 为什么选择 Xcompute

1. 一个 API Key 即可访问多种主流模型。
2. 兼容 OpenAI 接口协议，迁移成本低。
3. 支持文本、图像和视频等多种生成能力。
4. 提供渠道分组和计费模式，便于控制质量与成本。
5. 控制台、新版接口总览和模型定价页面配套完整，上手更快。

## 接入流程

<Steps>
  <Step title="创建 API Key">
    在控制台创建专属密钥，并为不同业务设置名称、额度或有效期。
  </Step>
  <Step title="选择模型与计费方式">
    根据任务场景选择模型、渠道分组以及按量或按次计费。
  </Step>
  <Step title="发送第一条请求">
    使用兼容 OpenAI 的接口地址与请求格式完成接入。
  </Step>
</Steps>

<Tip>
  如果你已经有基于 OpenAI 接口协议的代码，通常只需要替换 `Base URL` 和 `API Key`。
</Tip>
