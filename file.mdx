Xcompute 是一站式 AI 大模型 API 聚合平台。你可以用一个兼容 OpenAI 的接口，接入文本、图像和视频模型，并把能力快速集成到你自己的产品中。

<Info>
  Xcompute 面向开发者、团队和 AI 应用创业者。你可以在不改动整体调用方式的前提下，快速切换模型、统一管理密钥，并按场景优化成本。
</Info>

## 为什么选择 Xcompute

1. 一个 API Key 即可访问多种主流模型。
2. 兼容 OpenAI 接口协议，迁移成本低。
3. 支持文本、图像和视频等多种生成能力。
4. 提供渠道分组和计费模式，便于控制质量与成本。
5. 控制台、API 参考和模型定价页面配套完整，上手更快。

## 接入流程

<Steps>
  <Step title="创建 API Key">
    在控制台创建专属密钥，并为不同业务设置名称、额度或有效期。
  </Step>
  <Step title="选择模型与计费方式">
    根据任务场景选择模型、渠道分组以及按量或按次计费。
  </Step>
  <Step title="发送第一条请求">
    使用兼容 OpenAI 的接口地址与请求格式完成接入。
  </Step>
</Steps>

<Tip>
  如果你已经有基于 OpenAI 接口协议的代码，通常只需要替换 `Base URL` 和 `API Key`。
</Tip>
