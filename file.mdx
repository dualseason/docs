# Xcompute API 文档

> 一站式 AI API 中转平台，兼容 OpenAI 接口规范

## 快速开始

### 1. 获取 API Key

登录 [Xcompute 控制台](https://xcompute.us) → 个人中心 → API Keys → 创建新密钥

### 2. 基础配置

Base URL 选择与你使用的站点对应的地址：

| 站点 | Base URL | 说明 |
| --- | --- | --- |
| 国际站 | [`https://api.xcompute.us/v1`](https://api.xcompute.us/v1) | 推荐 |
| 加速入口 | [`https://intl.dualseason.com/v1`](https://intl.dualseason.com/v1) | 国内访问国际站时使用 |
| 国内站 | [`https://xcompute.dualseason.com/v1`](https://xcompute.dualseason.com/v1) | 不推荐 |
